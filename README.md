Setup
To run this project, install it locally using npm:
$ npm install
Setup .env example
Create .env file in your root project folder.

DB_HOST = localhost
DB_USER = root
DB_PASSWORD = your_password
DB_DATABASE = your_database
PORT = 4000
BASE_URL = http://localhost:4000/
SECRET_KEY = your-secret
Run the app
Development mode

$ npm run dev
Deploy mode

$ npm start
